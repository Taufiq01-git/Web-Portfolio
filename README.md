# taufiq. — Portfolio Web

Personal portfolio for Rahmat Taufiqurrahman — Product Designer.
Static site (HTML + CSS), responsive, built to deploy on Netlify.

## Structure
- `index.html` — home page (v.03 "Converting Cut")
- `style.css` — all styles + responsive rules (design tokens mirror the Design System)
- `assets/` — images (hero & about portraits, case visuals)

## Run locally
Just open `index.html` in a browser. No build step.

## Deploy
Hosted on Netlify (connected to this repo — every push auto-deploys).
